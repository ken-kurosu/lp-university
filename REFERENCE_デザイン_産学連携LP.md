## 1. 全体のデザイン方針

| 項目 | 内容 |
|------|------|
| **トーン** | 清潔感・プロフェッショナル。信頼性と説得力を強調。 |
| **レイアウト** | 縦長LP。セクションごとに背景色・レイアウトを変えて区切り。 |
| **カラーパレット** | **紺 / ライトブルー / 白 / 緑（CTA）**。黒背景のCTAセクションで緑を強調。 |
| **フォント** | モダンなサンセリフ（ゴシック）。見出しは大きく、本文は読みやすいサイズで統一。 |
| **背景のバリエーション** | 白 / ライトブルー（グラデ・パターン） / 淡い黄（グラデ） / **黒（CTA専用）** |

## 2. セクション別デザインの要点

### ヘッダー / ヒーロー
- **最上部:** 紺色の細いバー（ナビ・連絡先など小テキスト用）。
- **メインビジュアル:** 青空＋大学キャンパスなど、知的・信頼感のある写真。
- **キャッチ:** 白い大きな文字。「産学連携プロジェクトを始めたい…必見！」など。
- **サブコピー:** その下に小さめの説明。
- **CTA:**
  - 紺地＋白文字のボタン（「LINEで無料相談」「資料請求」など）を横並び。
  - その下に **メインCTA:** 明るい緑の大きいボタン「LINE登録で無料動画を受け取る」＋右向き白矢印。コンバージョン用として目立たせる。

※別バージョン: 「新時代の売り方」「企業×大学？」など白文字＋緑〜黄のアウトライン。握手のビジュアル重ね。

---

### 導入コンテンツ（淡い黄〜金系グラデ）
- ほんのり黄色がかったグラデで温かみ・高級感。
- 左: LINEアイコン・チェックマーク付きの箇条書き。
- 右: 動画サムネ（「大学教授の解説」など）またはビジネス写真。
- **比較ボックス（2つ並び）:**
  - 「一般的な産学連携」: グレー背景、錠アイコン、赤文字で「高コスト」「時間」などネガティブ要素。
  - 「私たちの産学連携」: 青背景、ロケット等アイコン、青文字で「低コスト」「スピード」などポジティブ要素。左右で明確に対比。

---

### CTAセクション（黒背景）
- ソリッドな黒背景のみ。
- 緑のCTAボタン1つ。高コントラストで強く目立たせる。
- ページ中〜下で複数回繰り返し配置。

---

### フロー（ライトブルー背景）
- 薄い水色背景。上向き矢印など薄い図形で「流れ・進捗」を表現。
- **見出し:** 「産学連携プロジェクト開始までの流れ」など。
- **ステップ:** 1・2・3 を横並び。各ステップに「アイコン（電話・PC・握手等）＋短いタイトル＋説明文」。

---

### 実績 / 事例（白背景）
- **見出し:** 「産学連携によって生まれた商品・サービス」など。
- **グリッド:** 3×3のカード。各カード: 写真＋年号＋タイトル＋短い説明＋「商品を見る」などの小ボタン。
- 別パターン: 「共同研究をした方から」お客様の声。カードに半透明の白ボックス＋赤タイトル＋引用。左上に「01」「02」…の緑数字。

---

### 協力大学 / 提携大学（白背景）
- **見出し:** 「協力大学」または「提携大学の紹介」。
- 各大学: **青い帯で大学名** → その下に建物・キャンパス写真 → 説明文。ブロックを積み上げ。

---

### メリット / 選ばれる理由（ライトブルー背景）
- 再び薄い水色背景。
- **見出し:** 「選ばれる4つの理由」など。
- **2×2グリッド:** 各ボックスにアイコン＋「POINT 01: 独自コネクション」など＋短い説明。
- その下に人物イラスト（対象者・関係者イメージ）を配置するパターンも。

---

### 最終CTA＋フッター
- 「まずはLINE登録後、無料動画と詳細をお受け取りください。」を大きく。
- 4つの小アイコン＋短い説明で「登録後の流れ・メリット」を視覚化。
- 緑の「LINEで詳細を確認する」ボタン（LINEアイコン付き）を中央に。
- **フッター:** 白〜うっすらライトブルー。「よくある質問」「お問い合わせ」「プライバシーポリシー」「特定商取引法」などリンク＋著作権表記＋「TOPへ」。

## 3. コンポーネント・パーツの統一

| 要素 | デザイン |
|------|----------|
| **メインCTA** | 鮮やかな緑、角丸。「LINEで詳細を確認する」または「LINE登録で無料動画を受け取る」＋LINEアイコン（＋右向き矢印）。 |
| **サブCTA** | 紺背景＋白文字のボタン。複数ある場合は横並び。 |
| **見出し** | 大きく、ゴシック。必要に応じて青やグラデのアクセント。 |
| **番号・ステップ** | 緑の数字や、青い「POINT 01」形式。アイコンとセットで。 |
| **カード** | 白または薄い色の背景、影または枠で軽く区切り。写真＋テキスト＋小ボタンの組み合わせ。 |
| **比較** | 左ネガティブ（グレー・赤）、右ポジティブ（青）で対比をはっきり。 |

---

## 4. ビジュアルの使い方

- **写真:** 大学の建物・キャンパス、ビジネスシーン、握手・打ち合わせなど。高品質で信頼感。
- **アイコン:** 線画でシンプル（電話、PC、握手、錠、ロケット、書類、ターゲットなど）。意味がひと目で分かるもの。
- **装飾:** 上昇矢印、波やグラデで「成長・進歩」を表現。薄い水色ベースで統一。
- **一貫性:** CTAの色・形・文言を全ページで統一。握手・矢印などのモチーフを繰り返してメッセージを強化。
