# レイアウト・パララックス効果の参照

**参照先URL:** https://labeauteonline.jp/lp/salon-keiei-school/

**用途:** 新LPの **レイアウトの組み方** と **パララックス効果の使い方** の参考にする。

※ページの自動取得はタイムアウトしたため、実装時はブラウザで該当URLを開き、スクロールやセクションの動きを確認してからデザインに反映すること。

---

## 参照する観点（実装・確認時に見るポイント）

### レイアウト
- セクションの**余白・幅・縦のリズム**（見出し〜本文〜次のセクションまでの間隔）
- **カラム構成**（1カラム / 2カラムの切り替え、画像とテキストの左右配置）
- **コンテンツのブロック分け**（どこで背景色を変えているか、カードやボックスの使い方）
- **ヒーロー〜導入〜本論〜CTA** の流れでのレイアウトの変化
- スマホ・タブレットでの**折り返し・縦積み**のイメージ（必要ならレスポンシブの参考にも）

### パララックス効果
- **スクロールに連動する動き**（背景画像の遅れ、奥行き感）
- **どのセクションで使っているか**（ヒーロー、区切り、フッター手前など）
- **動く要素**（背景のみ / 前景のイラストや図形も / テキストのフェードインなど）
- **速度・強さ**（控えめか、はっきりわかる程度か）
- 実装方法のイメージ（CSS `background-attachment: fixed`、`transform` + スクロール量、JS の scroll イベントなど）
