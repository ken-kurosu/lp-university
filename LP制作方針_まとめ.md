# 今回作っていきたいLPの方向性（まとめ）

これまで共有いただいた内容をもとに、「どんなLPを作っていきたいか」を整理したものです。

---

## 1. 基本方針

| 項目 | 参照元 | 方針 |
|------|--------|------|
| **訴求・テキスト・構成** | 既存LP（v2・黄・白ベース） | そのまま活かす。メッセージの流れとコピーをベースにする。 |
| **デザイン・ビジュアル** | 送付いただいた画像（青・白・緑のプロ仕様LP） | こちらを参考にする。清潔感・信頼感のある見た目に。 |

**一言でいうと:**  
「**訴求と中身は既存LP、見た目は今回のデザイン参考**」で、1本の新しいLPを作る。

---

## 2. 訴求・メッセージの流れ（既存LPベース）

1. **ヒーロー**  
   「自社商品×大学共同研究 産学連携プロジェクト」／「あなたのサービスを、差別化要素として打ち出しませんか？」  
   → LINE誘導のCTA

2. **課題提起**  
   「あなたの商品に、説得力はあるか？」  
   → 9割が埋もれる現状 → 選ばれる基準はエビデンス → 共同研究の声 → 「唯一無二の商品/サービス」「価格で争わなくてよい」へつなぐ

3. **ソリューション**  
   「これまで大企業の特権だった 産学連携」  
   → 従来は高コスト・長時間 → 当社は比較的安価に提供 → 「謳ってるだけ」から「語れる商品」へ → 「ぜひ大学と連携させてみませんか？」

4. **メリット**  
   「大学共同研究で得られる 5大メリット」  
   → 証明・唯一無二・現実的コスト・PR・ブランディング・世の中の商品にスポットライト、など（既存の6項目を活かす）

5. **最終CTA**  
   「まずはLINE登録後、無料動画と詳細をお受け取りください。」  
   → LINEで詳細を確認する

6. **フッター**  
   ©株式会社HUUK、プライバシーポリシー、特商法、TOPへ

※上記の**文言・順番・論理の流れ**は既存LPを尊重し、新LPでも維持する。

---

## 3. デザイン・見た目（参考画像ベース）

- **トーン:** 清潔感・プロフェッショナル。信頼性と説得力を視覚でも伝える。
- **カラー:** 紺 / ライトブルー / 白 / **緑（CTA）**。CTA専用で**黒背景**のブロックを挟み、緑ボタンを目立たせる。
- **レイアウト:** 縦長。セクションごとに背景色（白・ライトブルー・淡い黄グラデ・黒）を変えて区切る。
- **ヒーロー:** 紺の細いバー → 青空・キャンパスなど信頼感のあるビジュアル → 白い大きなキャッチ → 紺のサブCTA＋**緑のメインCTA**（LINE登録で無料動画など）。
- **導入:** 淡い黄〜金のグラデ。左に箇条書き、右に動画サムネ or 写真。**比較ボックス**（一般的な産学連携 vs 私たちの産学連携）で対比をはっきり。
- **フロー:** ライトブルー背景。「産学連携プロジェクト開始までの流れ」など、1・2・3のステップ＋アイコンで横並び。
- **実績・事例:** 白背景。3×3グリッドのカード（写真＋年号＋説明＋小ボタン）またはお客様の声カード。
- **協力大学:** 青い帯で大学名 → 写真 → 説明のブロックを積み上げ。
- **選ばれる理由・メリット:** ライトブルー背景。2×2などで「POINT 01」形式＋アイコン＋短い説明。
- **CTA:** 黒背景＋緑ボタンのみ。ページ中〜下で複数回。
- **フッター:** 白〜薄いライトブルー。リンク・著作権・TOPへ。

※上記は**見た目・コンポーネント・色・余白のトーン**の参考。訴求の「中身」は既存LPの構成・テキストに合わせる。

---

## 4. 新LPで実現したいこと（整理）

- **伝えたいこと（中身）**  
  既存LPと同じ。「差別化・説得力・エビデンス」「産学連携を比較的安価に」「5大メリット」「LINEで無料動画・詳細」という流れを崩さない。

- **見せ方（デザイン）**  
  参考画像のように、紺・ライトブルー・白・緑・黒を使い、清潔で信頼感のあるプロ仕様のLPにする。比較ボックス・フロー・事例グリッド・協力大学ブロックなど、構成要素も参考にする。

- **CTA**  
  LINE誘導で統一。緑ボタン＋「LINEで詳細を確認する」または「LINE登録で無料動画を受け取る」など、既存の訴求に合わせつつ、デザイン参考の「黒背景＋緑」で目立たせる。
